## ğŸš€ Getting Started

Follow these steps to run GhostTalk on your local machine.

### Prerequisites
- **PHP** installed on your system.
- A modern web browser.

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/ghosttalk-php.git
    cd ghosttalk-php
    ```

2.  **Start the Server**
    Double-click the `start_server.bat` file, or run the following command in your terminal:
    ```bash
    php -S 0.0.0.0:8000
    ```

3.  **Access the App**
    - **Local**: Open [http://localhost:8000](http://localhost:8000)
    - **Network**: Open `http://<YOUR_IP_ADDRESS>:8000` on other devices.
